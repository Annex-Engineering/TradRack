# 9g servo parts

The parts in this folder allow using a 9g/SG90 servo in place of the FEETECH MINI FT1117M. A metal gear servo is recommended.

## Replaced parts

The parts in this folder replace the following parts:

- STLs:
  - Selector:
    - `microfit_holder_x1`
    - `carriage_right_x1`
    - `servo_horn_x1`
